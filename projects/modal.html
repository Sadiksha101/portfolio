Ramro Designs

Ramro Designs (/rāmrō/), meaning "Beautiful Designs" in my native language Nepali, is an innovative platform created for the rug, carpet, and textile industry. It serves as a hub for inspiring design collections, providing valuable resources for both established professionals and aspiring designers. In collaboration with a colleague, I contributed to building the frontend of this platform for Alternative Technology Pvt. Ltd.

ReactJS Tailwind CSS Redux

What I learned and used with ReactJS:

  • Component-based architecture for building reusable UI components.
  • Using react hooks.
  • Routing with React Router for seamless navigation.

What I learned and used with CSS:

  • Utility-first CSS framework for building custom designs quickly.
  • Responsive design using Tailwind's built-in breakpoints.
  • Using Tailwind for consistent spacing, colors, and typography.
  • Customization of themes through the Tailwind config file.

What I learned and used with Tailwind:

  • Utility-first CSS framework for building custom designs quickly.
  • Responsive design using Tailwind's built-in breakpoints.
  • Using Tailwind for consistent spacing, colors, and typography.
  • Customization of themes through the Tailwind config file.

What I learned and used with Redux:

  • Centralized state management for handling global app state.
  • Using Redux Toolkit for creating actions and reducers efficiently.
  • Async actions using `createAsyncThunk` for API calls.
  • Integrating Redux with React components for state updates.
